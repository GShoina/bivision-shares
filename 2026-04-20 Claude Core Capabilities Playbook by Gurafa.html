1. Lead Scraping — როგორ Claude-ით

მექანიზმი

Claude-ს პირდაპირ web-ზე წვდომა აქვს 3 გზით:

  • MCP server (external service): Firecrawl MCP, eGet MCP, Exa MCP — structured extraction, batch, LLM post-processing
  • Playwright MCP (browser automation): JS-rendered pages, auth-walled content, LinkedIn/FB-ის behind-login
  • WebFetch + WebSearch (built-in): public docs, blog, domain audit, HTML parsing curl-ით

2. Lead Scoring (ICP Match) — როგორ Claude-ით

მექანიზმი

GTM Repository pattern (2026 standard): markdown ფაილების ნაკრები git repo-ში. Claude Code კითხულობს, ქულიანდ-ს აძლევს.

  1. icp/definition.md — ვინ არის ideal client (industry, size, tech stack, pain)
  2. icp/signal-library.md — რას ვხედავთ როგორც positive/negative signal (e.g., "hires BI analyst" = +10, "legacy SAP" = +5, "Odoo ERP-ით კმაყოფილი" = -8)

3. Lead Magnet Generation — როგორ Claude-ით

მექანიზმი

Claude-ს "artifacts-builder" pattern: Claude Code-მ React/Tailwind/shadcn ან plain HTML + JS-ით interactive calculator, assessment, quiz, mini-tool 5-30 წუთში აგებს. gated email-ით (ContactForm → HubSpot).

3 tier:

  1. Static calculator — 1 HTML file, localStorage, simple math (ROI, payback, cost)
  2. Assessment / quiz — 10-15 კითხვა → score + PDF report Claude-generated
  3. Micro-SaaS free tool — fully functional mini-app (e.g., "BI dashboard template builder", saves to localStorage)

4. Copywriting (Bivision voice) — როგორ Claude-ით

მექანიზმი

3-layer skill stack — Claude-ს რომ Bivision-ისავით დაწეროს, არა generic ChatGPT voice-ით:

  1. Brand voice skill — skills/bivision-voice/SKILL.md + voice-guide.md + examples/ (20-50 გვერდი existing copy: Proposal v2, client emails, LinkedIn posts, Owner-ის podcast/interview თუ არის)
  2. Content-type skill — skills/linkedin-post/, skills/proposal/, skills/cold-outreach/, თითო თავისი structure-ით
  3. SEO/angle skill — keyword list, pain language map, internal link strategy
