// ============= CV per Payer Source =============
// Coefficient of Variation = Stdev / Avg × 100 — relative volatility

// Master Measure — CV per Source (dim = Source)
Stdev(Aggr(Sum(Amount), Period))
  /

  * 100

📐 Working Capital Cushion — Detailed Calculation

Cash / Avg Monthly Fixed Cost — სრული დათვლის ლოგიკა Qlik analyst-ისთვის

Step-by-step

Step 1 — Fixed Cost Categories (8 categories)
Salaries · Rent · Utilities · Software/IT · Repair · Services Outsourced · Taxes · Supplies
Step 2 — Sum Per Month
FixedCost(month) = Salaries + Rent + Utilities + Software + Repair + Services + Taxes + Supplies
Step 3 — Average Across All Periods
AvgFixed = mean(FixedCost across 17 months) = 1,270,325 ₾
Step 4 — Latest Cash Position
LatestCash = Sum(ClosingCash, May-26) = 375,243 ₾
Final — Working Capital Cushion (months)
375,243 ÷ 1,270,325 = 0.30 თვე (≈ 9 დღე fixed cost buffer)
⚠️ Interpretation: May-26-ში არსებული ფული ფარავს მხოლოდ ~9 დღის ფიქსირებული ხარჯს (ხელფასი, რენტი, კომუნალურები, IT, რემონტი, აუთსორსი, გადასახადები, კანცელარია). თუ შემოსავალი 30 დღით დააგვიანდება — ხელფასი ვერ გადაიხდება.

Qlik Implementation

// ============= Working Capital Cushion =============

// Step A: Define fixed cost categories (in data model or as expression)
// Recommended: add "ExpenseType" field with values "Fixed"/"Variable" in load script
LOAD Period, Account, Amount,
  IF(Match(Account,
    'Salaries', 'Rent', 'Utilities', 'Software',
    'Repair', 'Services', 'Taxes', 'Supplies'
  ), 'Fixed', 'Variable') as ExpenseType
FROM CashFlow;

// Step B: Monthly fixed cost (master measure)
Sum({<ExpenseType={'Fixed'}>} Abs(Amount))

// Step C: Average monthly fixed cost across the period
Avg(Aggr(
  Sum({<ExpenseType={'Fixed'}>} Abs(Amount)),
  Period
))

// Step D: Working Capital Cushion (months) — final master measure
Sum({<Period={"$(=Max(Period))"}>} ClosingCash)
  /
Avg(Aggr(
  Sum({<ExpenseType={'Fixed'}>} Abs(Amount)),
  Period
))

// Step E: Alternative — cushion in DAYS (×30)
Sum({<Period={"$(=Max(Period))"}>} ClosingCash)
  /
(Avg(Aggr(
  Sum({<ExpenseType={'Fixed'}>} Abs(Amount)),
  Period
)) / 30)

📋 Methodology + Caveats

  • Source: Direct CF.xls — HTML-as-XLS ERP export (159KB)
  • Period: Jan-25 → May-26 (17 months active) + Dec-24 baseline. Jun-26 column empty.
  • "Revenue" = Cash Receipts (cash basis, არა accrual P&L). Qlik analyst-ი source-ში გადაამოწმოს თუ accrual revenue-ც ხელმისაწვდომია.
  • Sign convention: Receipts positive, expenses stored negative — Qlik formula-ში Abs() ratio-ებისთვის.
  • Currency: assumed GEL (₾) — source-ში არ არის currency code.
