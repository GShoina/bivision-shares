4. Setup Runbook — 5 წუთი

Option A — VS Code (Recommended)

  1. Install VS Code + GitHub Copilot Chat extension
  2. Install Power BI Modeling MCP VS Code extension
  3. Copilot chat → confirm powerbi-modeling-mcp ხელმისაწვდომია

Option B — Claude Code / Manual

Add to MCP client config (npx auto-downloads):

{
  "powerbi-modeling-mcp": {
    "type": "stdio",
    "command": "npx",
    "args": [
      "-y",
      "@microsoft/powerbi-modeling-mcp@latest",
      "--start"
    ]
  }
}

Prerequisites: Node.js · Power BI Desktop OR Fabric workspace OR PBIP folder
