4. Setup Runbook — 5 წუთი

Option A — VS Code (Recommended)

  1. Install VS Code + GitHub Copilot Chat extension
  2. Install Power BI Modeling MCP VS Code extension
  3. Copilot chat → confirm powerbi-modeling-mcp ხელმისაწვდომია

Option B — Claude Code / Manual

Add to MCP client config (npx auto-downloads):

{
  "powerbi-modeling-mcp": {
    "type": "stdio",
    "command": "npx",
    "args": [
      "-y",
      "@microsoft/powerbi-modeling-mcp@latest",
      "--start"
    ]
  }
}

Prerequisites: Node.js · Power BI Desktop OR Fabric workspace OR PBIP folder

4. რა შემიძლია გავაკეთო — Example Scenarios

  • Bulk rename hundreds of objects ერთი prompt-ით: "Analyze naming convention of 'Sales' table and apply pattern across model"
  • Auto-document model: "Add descriptions to all measures, columns, tables explaining purpose and DAX logic"
  • Multilingual translation: "Generate French translation for model" (Georgian-ად მუშავდება ✓)
  • Refactor measures → Calculation Groups: "Refactor 'Sales 12M Avg' and '6M Avg' into calculation group + add 24M, 3M variants"
  • Power Query parameterization: "Analyze Power Query, identify data source, create parameters for easy switching"
  • Benchmark DAX across versions: "Connect V1 + V2 models, benchmark [query]"
  • Generate Markdown documentation + Mermaid relationship diagram + business-friendly descriptions
  • RLS setup — security roles ბუნებრივი ენით
